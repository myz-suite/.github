# MyZ Suite

MyZ Annotator 是一款 Chrome 扩展，帮助你在任意网页高亮文本、撰写批注、截图收集，并在本地管理这些资料。

## 安装链接

- **Chrome**: [MyZ Annotator on Chrome Web Store](https://chromewebstore.google.com/detail/myz-annotator/mhakfcbobhdemicjelhjjpmgibhnnplg)
- **Edge**: [MyZ Annotator on Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/myz-annotator/fepakkoggnancnpcclldodfbpgobkabj)
